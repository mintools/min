Query Syntax:
abcfinds tasks that include "abc" in the specified field
"xyz abc"finds tasks that include "xyz abc" in the specified field
xyz abcfinds tasks that include "xyz" OR "abc" in the specified field
xyz AND abcfinds tasks that include "xyz" AND "abc" in the specified field
te?t? matches any one character
te*t* matches any number of character
Note: You cannot use a * or ? symbol as the first character of a search.
